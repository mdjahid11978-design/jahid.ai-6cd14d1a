# Jahid.ai

A responsive static landing page for Jahid.ai, designed as a polished starting point for an AI builder, consultant, or digital studio portfolio.

## What is included

- Modern hero section with calls to action
- Services, featured work, process, and contact sections
- Responsive navigation with a mobile menu
- Accessible skip link and semantic page structure
- Lightweight scroll/reveal interactions with reduced-motion support

## Project structure

```text
.
├── index.html   # Page content and semantic structure
├── styles.css   # Responsive visual design
├── script.js    # Navigation, header, and reveal interactions
├── robots.txt   # Search crawler defaults
├── sitemap.xml  # Basic sitemap placeholder
└── LICENSE
```

## Run locally

Because this is a static site, you can open `index.html` directly or serve the folder with a local web server:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.

## Customize

Update the placeholder copy, project examples, and `mailto:hello@jahid.ai` contact link in `index.html` before publishing.
